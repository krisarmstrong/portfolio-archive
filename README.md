# Kris Armstrong GitHub Pages

![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-F7DF1E?logo=javascript&logoColor=black) ![License](https://img.shields.io/badge/License-MIT-green) ![npm](https://img.shields.io/badge/npm-package-CB3837?logo=npm) ![GitHub Pages](https://img.shields.io/badge/Deployed-GitHub%20Pages-222222?logo=github) ![Status](https://img.shields.io/badge/Status-Active-success)


Personal website and professional portfolio hosted on GitHub Pages.

## Description

A professional website showcasing career highlights, certifications, and contact information. Built with modern web technologies including Bootstrap 5, custom components, and responsive design principles.

## Features

- Responsive Bootstrap 5 design
- Dynamic component loading
- Professional summary and highlights
- Certifications showcase
- Contact information
- Dark theme interface
- Custom styling with Tailwind CSS
- Vite build system

## Tech Stack

- HTML5, CSS3, JavaScript
- Bootstrap 5.3
- Bootstrap Icons
- Google Fonts (Inter)
- Tailwind CSS 3
- Vite 6
- PostCSS

## Installation

```bash
git clone <repository-url>
cd krisarmstrong.github.io
npm install
```

## Usage

Development:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Project Structure

- `index.html` - Home page
- `about.html` - About page
- `certifications.html` - Certifications showcase
- `contact.html` - Contact information
- `404.html` - Custom 404 page
- `components/` - Reusable UI components
- `src/` - Source files and assets
- `styles/` - Custom CSS

## Deployment

This site is automatically deployed to GitHub Pages when changes are pushed to the main branch.
